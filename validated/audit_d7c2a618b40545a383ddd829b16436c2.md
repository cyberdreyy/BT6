### Title
Delegated Pusher Can Push Zero-Price Slot Word to Permanently Stall Pool Swaps — (`smart-contracts-poc/contracts/oracles/compressed/CompressedOracle.sol`)

### Summary

The `CompressedOracleV1.fallback()` push path accepts any slot word that carries a strictly newer timestamp, with **no validation that the encoded price data is non-zero**. A delegated pusher (semi-trusted) can push a word where all oracle lanes are zeroed (`p = 0, s0 = 0, s1 = 0`) but the timestamp is fresh. The slot is overwritten with zero prices and a valid timestamp, causing every subsequent `AnchoredPriceProvider` read for that feed to revert with `FeedStalled`, permanently blocking swaps for any pool bound to that feed until a legitimate pusher overwrites the slot again.

This is the direct analog of the external report's "repay zero resets borrow index" pattern: a zero-value but structurally valid operation resets critical state (the slot's price data) without providing useful information, and no guard prevents it.

---

### Finding Description

**Root cause — `CompressedOracle.sol` `fallback()`:** [1](#0-0) 

The only two checks applied to each incoming 32-byte slot word are:

1. `timestampMs.revertIfAfterBlockTimeWithDrift(MAX_TIME_DRIFT)` — timestamp must not be in the future.
2. `timestampMs.isAfter(oldTimestampMs)` — timestamp must be strictly newer than stored. [2](#0-1) 

There is **no check that the oracle lane data (`p`, `s0`, `s1`) is non-zero or non-sentinel**. The entire slot (all four positions) is then written verbatim:

```solidity
_writeStorage(key, bytes32(bytes32(word & ~uint256(0xff))));
```

A malicious pusher sends a word of the form:

```
[0x000000000000][0x000000000000][0x000000000000][0x000000000000][newer_ts:56][slotId:8]
```

All four oracle lanes are zero; the timestamp is newer than stored. The word passes both checks and is committed to storage.

**Decoding the zeroed slot:**

`U64x32.decode(0)` returns `0` (mantissa=0, exponent=0). [3](#0-2) 

`Codebook256.decode(0)` returns `(0, true)` — index 0 maps to spread value `0x0000 = 0 bps`, which is a **valid** (non-error) decode. [4](#0-3) 

So `getOracleData` for any position in the zeroed slot returns `price = 0, spread0 = 0, spread1 = 0, timestampMs = newer_ts` — a structurally valid but zero-price observation with a fresh timestamp. [5](#0-4) 

**Propagation to `AnchoredPriceProvider`:**

`_readLeg` checks `mid == 0` and returns `ok = false`: [6](#0-5) 

`getBidAndAskPrice` then reverts with `FeedStalled()`: [7](#0-6) 

Every swap attempt on any pool bound to that feed reverts for as long as the zeroed slot remains the most recent update.

---

### Impact Explanation

Any pool whose `AnchoredPriceProvider` reads from the corrupted feed cannot execute swaps. The `FeedStalled` revert is thrown on every `getBidAndAskPrice` call, making the pool's swap path completely unusable. This matches the allowed impact: **"Broken core pool functionality causing loss of funds or unusable withdraw/swap/liquidity flows."**

---

### Likelihood Explanation

The trigger is a **delegated pusher** — an address explicitly authorized by the feed creator via `allowPushers` or `allowContractPushers`. This is a semi-trusted role. A compromised or malicious pusher key can execute the attack in a single transaction with no additional privileges. The creator cannot prevent the attack in the same block; recovery requires the creator (or another pusher) to push a valid slot word with a yet-newer timestamp. [8](#0-7) 

---

### Recommendation

Add a non-zero price guard in the `fallback()` loop before writing. For each oracle lane present in the word, verify that the 32-bit price field (`p`) is non-zero, or that the lane carries the explicit stall sentinel (`s0 == 0xFF && s1 == 0xFF`). Reject (revert or skip) any word where a lane has `p == 0` without the sentinel, mirroring the same "zero-amount" guard the external report recommends for the lending protocol.

---

### Proof of Concept

```solidity
// Attacker is a delegated pusher for `creator`
// Pool uses feedIdOf(creator, slotIndex=0, positionIndex=0)

uint56 newerTs = uint56(block.timestamp * 1000); // fresh timestamp
uint8  slotId  = 0;

// All oracle lanes zero, only timestamp and slotId set
uint256 zeroWord = (uint256(newerTs) << 8) | uint256(slotId);
// p=0, s0=0, s1=0 for all four positions → lanes [255:64] are all 0

vm.prank(delegatedPusher);
(bool ok,) = address(oracle).call(abi.encodePacked(zeroWord));
assertTrue(ok); // passes: timestamp is newer, no price validation

// Now any pool swap reverts:
vm.expectRevert(AnchoredPriceProvider.FeedStalled.selector);
pool.swap(...);
```

The zeroed slot word satisfies both guards (`timestampMs` not future, `timestampMs` newer than stored) and is committed. `getOracleData` returns `price = 0` with a fresh timestamp. `AnchoredPriceProvider._readLeg` sees `mid == 0` and returns `ok = false`, causing `getBidAndAskPrice` to revert with `FeedStalled` on every subsequent swap.

    fallback() override external {
        uint256 end;
        uint256 namespace;

        address creator = namespaceRemapping[msg.sender];
        if (creator == address(0)) creator = msg.sender;

        assembly ("memory-safe") {
            end := calldatasize()
            namespace := shl(96, creator) // [creator:20][zeros:12]
        }

        // 4 * 6 + 7 + 1 = 32 bytes per slot
        if (end == 0 || end % 32 != 0) revert BadCalldataLength();

        for (uint256 ptr = 0; ptr < end; ptr += 32) {
            uint256 word;
            assembly ("memory-safe") {
                word := calldataload(ptr)
            }
            // casting to 'uint8' is safe we want LSB
            // forge-lint: disable-next-line(unsafe-typecast)
            uint8 slotId = uint8(word);
            TimeMs timestampMs = toTimeMs(word >> 8 & X56);
            timestampMs.revertIfAfterBlockTimeWithDrift(MAX_TIME_DRIFT);
            bytes32 key = bytes32(namespace | uint256(slotId));
            uint256 old = uint256(_loadStorage(key));
            TimeMs oldTimestampMs = toTimeMs(old >> 8 & X56);

            bool newer = timestampMs.isAfter(oldTimestampMs);
            if (!newer) continue;

            _writeStorage(key, bytes32(bytes32(word & ~uint256(0xff))));

    function _readLeg(bytes32 feedId)
        internal returns (uint256 mid, uint256 spreadBps, uint256 refTime, bool ok)
    {
        (mid, spreadBps, , refTime) = IPricedOracle(address(offchainOracle)).price(feedId, msg.sender);

        // Stale reference → not ok. Clamping to a stale anchor is the one false-safety case.
        if (_isStale(refTime, block.timestamp, MAX_REF_STALENESS)) return (mid, spreadBps, refTime, false);

        // Basic validity — mid positive, spreadBps not the stalled/off-hours marker (the Chainlink oracle
        // writes spreadBps = ORACLE_BPS when an RWA market is closed).
        if (mid == 0 || spreadBps >= ORACLE_BPS) return (mid, spreadBps, refTime, false);

        // Per-leg price guard.
        (uint128 guardMin, uint128 guardMax) = offchainOracle.priceGuard(feedId);
        guardMax = guardMax == 0 ? type(uint128).max : guardMax;
        if (mid < guardMin || mid > guardMax) return (mid, spreadBps, refTime, false);

        ok = true;
    }
